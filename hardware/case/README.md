# 3D Printed Enclosure

STL files for the blinds controller enclosure.

## Files

| File | Description | Size |
|------|-------------|------|
| `enclosure_bottom.stl` | Main body / base | 619 KB |
| `enclosure_top.stl` | Lid / cover | 245 KB |

## Print Settings (Suggested)

| Setting | Value |
|---------|-------|
| Layer Height | 0.2mm |
| Infill | 15-20% |
| Supports | As needed (check overhangs) |
| Material | PLA or PETG |

## Assembly

1. Print both parts
2. Install PCB in bottom enclosure
3. Connect battery and motor wires
4. Snap or screw on the top lid

## Notes

- Check fit with your specific PCB revision
- May need to adjust for different battery sizes
- Ventilation holes recommended if running for extended periods
